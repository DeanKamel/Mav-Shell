# Mav-Shell
Shell program using C 

## Features
- Runs and executes shell supported commands.
 
 ## How to run
 - Using a VM:
 - Download and extract file
 - Change the working directory to where you have your C program. (cd)
 - Type in: "gcc msh.c"
 - To run the executable: "./a.out"
